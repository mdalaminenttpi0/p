# URL to PDF Converter 📄

একটি শক্তিশালী ওয়েব টুল যা যেকোনো ওয়েবসাইটকে উচ্চ মানের PDF-এ রূপান্তরিত করে। **HD, 4K এবং 8K UHD** রেজোলিউশন সমর্থন করে এবং মোবাইল-বান্ধব ডিজাইন সহ।

## 🌟 বৈশিষ্ট্য

✅ **একাধিক রেজোলিউশন**
- HD (1920×1080)
- 4K (3840×2160)
- 8K UHD (7680×4320)

✅ **মোবাইল-বান্ধব**
- প্রতিক্রিয়াশীল ডিজাইন
- সব ডিভাইসে কাজ করে
- স্পর্শ-অনুকূল ইন্টারফেস

✅ **উন্নত বৈশিষ্ট্য**
- রূপান্তর ইতিহাস সংরক্ষণ
- মেটাডেটা সমর্থন
- ত্রুটি পরিচালনা
- অগ্রগতি ট্র্যাকিং

✅ **নিরাপদ এবং দ্রুত**
- কোনো ডেটা সংরক্ষণ নেই
- সার্ভার-সাইড প্রসেসিং
- সীমিত টাইমআউট

## 🚀 শুরু করুন

### প্রয়োজনীয়তা
- Node.js (v14 বা উচ্চতর)
- npm বা yarn

### ইনস্টলেশন

```bash
# রেপোজিটরি ক্লোন করুন
git clone https://github.com/mdalaminenttpi0/p.git
cd p

# নির্ভরতা ইনস্টল করুন
npm install

# .env ফাইল তৈরি করুন
cp .env.example .env

# সার্ভার চালু করুন
npm start
```

### ব্যবহার

1. ব্রাউজারে খুলুন: `http://localhost:5000`
2. ওয়েবসাইটের URL প্রবেশ করুন
3. পছন্দের রেজোলিউশন নির্বাচন করুন (HD/4K/8K)
4. **Convert to PDF** বোতাম ক্লিক করুন
5. PDF ডাউনলোড হবে

## 🔌 API এন্ডপয়েন্ট

### 1. স্বাস্থ্য পরীক্ষা
```bash
GET /api/health
```

### 2. PDF তে রূপান্তর করুন
```bash
POST /api/convert-to-pdf
Content-Type: application/json

{
  "url": "https://example.com",
  "quality": "8k"
}
```

### 3. উন্নত রূপান্তর (স্ক্রিনশট সহ)
```bash
POST /api/convert-with-screenshot
Content-Type: application/json

{
  "url": "https://example.com",
  "quality": "8k",
  "includeMetadata": true
}
```

### 4. ব্যাচ রূপান্তর
```bash
POST /api/batch-convert
Content-Type: application/json

{
  "urls": ["https://site1.com", "https://site2.com"],
  "quality": "4k"
}
```


## ⚙️ কনফিগারেশন

`.env` ফাইলে সেটিংস কাস্টমাইজ করুন:

```env
PORT=5000                    # সার্ভার পোর্ট
NODE_ENV=development         # পরিবেশ (development/production)
PDF_TIMEOUT=30000           # রূপান্তর টাইমআউট (মিলিসেকেন্ড)
PDF_QUALITY_DEFAULT=hd      # ডিফল্ট রেজোলিউশন
```
